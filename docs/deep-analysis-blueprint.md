# 股市深度分析 — 產品藍圖

> 本文件規劃將現有 CashFlow 從「現金流模擬 + 淺層市場快照」擴充為一個可做**深度股市分析**的平台。
> 方向:**基本面分析 + 籌碼/資金面分析 + AI/量化策略**,資料來源優先採用**免費真實資料**。
> 本專案僅供教育與產品展示,非投資建議。

---

## 1. 產品定位

| 項目 | 現況 (CashFlow) | 目標 |
| --- | --- | --- |
| 市場資料 | 即時報價快照 + 合成假趨勢 | 真實歷史 + 財報 + 籌碼 |
| 分析深度 | 一條簡單訊號規則 | 三大分析模組 + 綜合評分 |
| 標的 | 台股/美股 ETF・科技股 | 台股個股為主、美股輔助 |
| AI | 報價問答 | 結合真實數據的分析與回測 |

**重點**:籌碼面(法人買賣、融資融券、外資持股)基本上是**台股特有**且最有資訊優勢的領域,所以台股個股會是核心,美股以基本面為主。

## 2. 三大分析模組

### 2.1 基本面分析 (Fundamental)

**台股 (FinMind) / 美股 (yfinance) 都涵蓋**

- 財務報表:損益表、資產負債表、現金流量表(近 8 季 / 5 年)
- 獲利能力:EPS、毛利率、營益率、淨利率、ROE、ROA
- 成長性:營收 YoY/MoM、EPS 成長、近 12 個月營收動能
- 估值:本益比 (PER)、股價淨值比 (PBR)、殖利率、PEG
- 財務健康:負債比、流動比、自由現金流、利息保障倍數
- **估值模型**:DCF(現金流折現)、本益比河流圖、合理價區間
- **綜合評分**:把上述指標標準化後給 0–100 的基本面分數

### 2.2 籌碼/資金面分析 (Chip / Fund Flow) — 台股核心

**FinMind 免費提供:**

- 三大法人買賣超(外資、投信、自營商)— 逐日 + 累計
- 融資融券餘額與增減(散戶情緒指標)
- 外資持股比率變化
- 集保戶股權分散(大戶 vs 散戶持股趨勢)
- 借券賣出餘額
- **衍生指標**:主力進出(估算)、法人連續買超天數、券資比、籌碼集中度
- **綜合評分**:籌碼面分數(法人態度 + 散戶反向指標 + 集中度)

### 2.3 AI / 量化策略 (Quant / AI)

- **回測引擎 (Backtesting)**:給定策略規則,跑歷史報酬、勝率、最大回撤、夏普值
- **因子分析**:價值/成長/動能/品質因子打分與排序
- **選股器 (Screener)**:用基本面 + 籌碼條件篩選股票池
- **AI 分析報告**:把三模組的真實數據餵給 LLM(沿用現有 OpenAI/Groq 整合),產生個股深度分析摘要與多空論點
- (進階) ML 預測:用歷史特徵做趨勢機率預測 — 列為後期選配

| 來源 | 成本 | 覆蓋 | 用途 |
| --- | --- | --- | --- |
| **FinMind** | 免費(有額度) | 台股全面 | 財報、法人、融資券、外資持股、股價 |

## 4. 後端架構 (FastAPI)

沿用現有 `backend/app/` 結構,新增:

```
backend/app/
├── services/
│   ├── data_providers/
│   │   ├── finmind_client.py      # FinMind 封裝 + 快取
│   │   └── yfinance_client.py     # yfinance 封裝
│   ├── fundamental_engine.py      # 基本面指標 + 評分 + DCF
│   ├── chip_engine.py             # 籌碼面指標 + 評分
│   ├── quant_engine.py            # 回測 + 因子 + 選股
│   └── analysis_aggregator.py     # 綜合三模組 → 總評分 + AI 報告
├── routes/
│   ├── fundamental.py             # /api/fundamental/{symbol}
│   ├── chip.py                    # /api/chip/{symbol}
│   ├── quant.py                   # /api/quant/backtest, /screener
│   └── analysis.py                # /api/analysis/{symbol} 綜合
```

**核心 API**

| 端點 | 回傳 |
| --- | --- |
| `GET /api/fundamental/{symbol}` | 財報、估值、成長、基本面評分 |
| `GET /api/chip/{symbol}` | 法人、融資券、外資持股、籌碼評分(台股) |
| `POST /api/quant/backtest` | 回測績效(報酬、勝率、回撤、夏普) |
| `POST /api/quant/screener` | 依條件篩股 |
| `GET /api/analysis/{symbol}` | 三模組整合 + 總評分 + AI 報告 |

## 5. 前端頁面 (Next.js)

沿用現有 App Router 與 Tailwind/Recharts,新增:

- `/stock/[symbol]` — **個股深度頁**(核心):分頁顯示基本面 / 籌碼面 / 量化 / AI 報告
- `/screener` — 選股器
- `/backtest` — 策略回測介面
- 現有 `/market` 升級為入口,可點進個股深度頁

**視覺化**:本益比河流圖、法人買賣超柱狀 + 股價疊圖、融資券雙軸圖、財報長條趨勢、回測權益曲線、雷達圖(三模組評分)。

## 7. 風險與注意事項

- **免費 API 額度**:FinMind 免費版有每日請求上限 → 必須快取 + 批次更新
- **資料正確性**:財報/籌碼有公告時間差,需標註資料日期
- **籌碼面僅台股**:美股無對等的免費法人買賣資料,個股頁需依市場切換可用模組
- **合規**:全程標註「非投資建議」(現有 README 已有此聲明,需延續到每個分析頁)
- **不執行交易**:本平台只做分析,不接券商下單
